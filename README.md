# Hate Speech Detection Project

Jupyter notebook analyzing 24K+ Twitter tweets to classify hate speech, offensive language, and neutral content using classical ML models.

## Features
- **Dataset**: 24,783 labeled tweets (hate_speech=0, offensive_language=1, neither=2) [file:1]
- **Preprocessing**: TF-IDF vectorization on tweet text [file:1]
- **Models**: Logistic Regression and DecisionTreeClassifier [file:1]
- **Evaluation**: Model performance metrics and confusion matrices [file:1]

## Quick Start
1. Clone: `git clone https://github.com/yourusername/hate-speech-detection.git`
2. Install: `pip install pandas scikit-learn jupyter`
3. Run: `jupyter notebook hatespeechproject.ipynb`
